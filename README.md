# Agrinho
# 🌱 AgroFuturo Sustentável

Landing page moderna com foco no agronegócio sustentável, destacando o equilíbrio entre produção agrícola, tecnologia e preservação ambiental.

---

## 📌 Sobre o Projeto

O **AgroFuturo Sustentável** é uma página front-end desenvolvida para representar um conceito de agronegócio moderno e responsável.

A proposta visual combina:

- 🌿 Sustentabilidade
- 🚜 Tecnologia no campo
- 💧 Preservação dos recursos naturais
- 🌎 Responsabilidade ambiental
- 📈 Alta produtividade

---

## 🎨 Características da Interface

- Design moderno e responsivo
- Hero section com imagem de impacto
- Paleta de cores verdes e naturais
- Cards informativos
- Estatísticas visuais
- Navegação fixa (navbar)
- Layout adaptável para mobile

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts
- Responsividade com Media Queries

## 🚀 Como Executar

1. Baixe ou clone o projeto:

```bash
git clone https://github.com/seu-usuario/agrofuturo.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo:

```bash
index.html
```

Ou utilize a extensão **Live Server** no VS Code.

## 🌎 Conceito

O projeto busca transmitir a ideia de que:

> “É possível produzir mais sem destruir o meio ambiente.”

A interface foi inspirada em tendências modernas de sites institucionais do agro e ESG.
